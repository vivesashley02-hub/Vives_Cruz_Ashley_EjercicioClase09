# Uso de IA

## Prompt 1
### Pregunta realizada

¿Cómo puedo aplicar SRP en la aplicación Tero Match?

### Respuesta resumida

Se indicó separar el código de la siguiente manera:

- FormularioMatch: muestra el formulario.
- ResultadoMatch: muestra el perfume recomendado.
- TarjetaPerfume: presenta la información de cada perfume.
- ListaGenerica: muestra diferentes tipos de elementos.
- useTeroMatch: contiene estados, validaciones y selección del perfume.
- perfumeServicio: guarda, consulta y elimina documentos de Firestore.

Esta separación permite que cada archivo tenga una responsabilidad principal.

## Prompt 2

### Pregunta realizada

¿Cómo publico una aplicación React creada con Vite en Firebase Hosting?

### Respuesta resumida

Se recomendó instalar Firebase CLI, iniciar sesión, ejecutar la configuración de Hosting, seleccionar la carpeta dist, construir la aplicación con npm run build y publicar utilizando firebase deploy.
